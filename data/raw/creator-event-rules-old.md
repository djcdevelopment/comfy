**Creator Event Etiquette & Rules**  
*React to the Event's Role in [**❓-how-to-join** to be notified for event reset announcements.](https://discord.com/channels/820120530107367435/820122587861352468)*

1. **Respecting Events:**   
   1. **No building** within events. No campfires, portals, workbenches, feasts with the serving tray, etc.   
   2. **No deconstructing** within events (by using a build hammer).   
   3. **No recoloring/uncoloring** build pieces within events.

2. **Unique Event Rules:** Follow all rules posted in-game for that particular event which may supersede the other rules on this list. 

   

3. **Magic (from Mistlands) Rules:** Magic (crafted from Mistlands materials) is allowed at Carapace events and above by default. This magic is **not allowed** at padded events and under by default. Magic from Mistlands is defined as anything crafted from that biome that requires eitr food.   
     
4. **Magic (from Ashlands) & Gem-Enhanced Weapon Rules**: Magic (crafted from Ashlands materials) and gem-enhanced weapons are allowed at Flametal events and above by default. They are **not allowed** at carapace events and under by default. Magic from Ashlands is defined as anything crafted from that biome that requires eitr food. 

5. **Clear the Enemies:** If the event has enemies, it is expected that all the enemies 	are killed to successfully complete the event and take all the loot. Exceptions in which this may not be possible are if 1\) enemies are godhealthed and not intended to be killed, 2\) there is a source of infinitely spawning enemies and 3\) enemies have glitched through the event and are not accessible as a player.   
     
6. **What Defines Full Group:** Defined by **minimum** amount of **recommended players** for the event listed in **\#🎪-creator-events**. To run an event solo, you must make a good faith effort to find a group \- see details in \#15.

7. **Who Claims the Event:** First full group that arrives in-game at the event; if a solo player arrives before a full group, they also claim it with the full group and **must run the event together**. **"First Time Thursdays"** for the first 6 weeks of an era make events you've already played off limits \- see \#16.

8. **Who can Join:** There is no obligation that once a full group has arrived that anyone outside of that is entitled to join (except if a solo player arrives first then they are entitled, see \#7), but if the existing group approves of additional players, that's allowed.

9. **Arrived at the Same Time:** In the event there is debate that the arrival of full groups was a close tie, 1 person from each party rolls for it and the **highest roller wins claiming the event.**

10. **Camping or Spamming Events:** Our two hard rules regarding this are  
    1. **Do not claim loot from a newly launched event more than twice during its first active week on the server and**  
    2. **Do not claim loot from an event two resets in a row according to its reset schedule posted in the [\#🔁-event-resets](https://discord.com/channels/820120530107367435/822500121726943232) channel.**

In general if you’ve already experienced an event several times, we encourage prioritizing letting those who haven’t run it have the run (if group size is an issue and you can’t play together)**.** It is also encouraged (optional) to make an effort to recruit other players who have not had that chance to run events.  

11. **Leaving Early:** Do not start the event if you aren't sure you can complete it. Events can take up to 2-3 hours. If you absolutely can't finish, **inform the Creator ASAP** via DM or opening a ticket in **\#❗-get-support** with them tagged. 

12. **Joining Late:** If a group has started the dungeon with a full group OR has made a good-faith effort to wait the requisite 15 minutes for groups less than the recommended minimum *(excluding solo)*, **it is at their discretion on whether a player who shows up may join their group.** Should they allow the player to join the group, they must split loot fairly amongst each other.

13. **Event Requests:** If you're struggling to be able to play an event before others clear it, experiencing significant lag in groups, or want to complete the event solo for a Jarl achievement (see \#15), Creators *can* be open to private reset requests via DMs, but it’s up to the Creator's discretion, and **it is expected to respect Creator's decision if they can't or decide not to.**  
      
14. **Event Feedback:** Play the event the way it was meant to be played. If you find bugs, shortcuts, or glitches, report it to the Creator so they have a chance to correct it. **You can contact a Creator with feedback via DMs.**

15. **How to Solo a Creator Event:**  
    1. **Guaranteed Solo from Start to Finish:** This can only be done by request with the Creator. Coordinate with the Creator of the event via DMs to see if they're willing to reset it for you to run it solo.  
    2. **Solo due to No Interested Parties:** Put in a good faith effort first. Good Faith Effort definition: Shout in-game chat and on discord in [\#🤝-looking-for-group](https://discord.gg/kqzSmjMn8N) using the @Events role. You must mention the specific event; "LFG for any event' doesn't count. If there's no interest after 15 minutes and there are no rules against soloing from the Creator (check the event post in discord or signs at the event), you're good to start solo. **However, with this method if people ask to join after you start, you can't turn them away.**

16. **First Time Thursdays**

Every Thursday for the **first 6 weeks of an era**, at least half the players in an event group must be players **who haven't run that event before** in the current era. **If you can't find or make a group with at least 50% people who haven't run it before, you sit out on running the event.** This allows new players to run the events, players to switch up who they run events with, and players who run events frequently to give others a chance. After Thursday has passed, if the event is still up, anyone can run it. After the first 6 weeks have passed, anyone can run any event on Thursdays.

**Starts:** Thursday  9AM CEST  //  3AM EST  //  12 AM PST  
**Ends:** Friday  9AM CEST  //  3AM EST  //  12 AM PST

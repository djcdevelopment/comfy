**Creator Event Etiquette & Rules**  
*React to the Event's Role in [**❓-how-to-join** to be notified for event reset announcements.](https://discord.com/channels/820120530107367435/820122587861352468)*

1. **Respecting Events:**   
   1. **Follow event-specific rules** (they may override/deviate from this list)  
   2. **No building, deconstructing, or recoloring/uncoloring** within events. Feasts can be “built” in event respawn areas / non combat only

2. **Allowed Gear:** All Magic is **not allowed** at Padded events and under by default. Ashlands Magic and Gemmed weapons are **not allowed** at Carapace events. Respect event-specific gear rules \- See Rule \#1a.  
     
3. **Completing an Event:**   
   1. **Clear the Enemies:** If the event has enemies, make a good faith effort to clear them all. You don't have to clear enemies that are inaccessible, godhealthed, or infinitely spawning.  
   2. **Loot Search:** All players should help search for loot. The event listing in **\#🎪-creator-events** may have info on the number of items to search for.  
   3. **Loot Split:** Loot should be split evenly, for singular items (uniques) players should roll and pick one at a time. (**\!roll 20** in the in-game chat)  
   4. **Leaving Early:** If you absolutely can't finish an event, **inform the Creator ASAP** via DM or opening a ticket in **\#❗-get-support**.

4. **Reset Requests:** Creators *may* be open to private reset requests via DMs, but it’s up to their discretion. **Respect the Creator’s decision.**

5. **Who Claims the Event:** First full group (Defined by **minimum** amount of **recommended players** for the event listed in **\#🎪-creator-events**) that arrives in-game at the event.  
   1. If a solo player arrives at the event first, they are entitled to join the run **even if a “full group” shows up.**  
   2. **"First Time Thursdays"** make events you've already played off limits \- see \#9a.  
   3. **Arrived at the Same Time:** If two full groups arrive at the same time, roll to decide which group gets to run. (**\!roll 20** in the chat box)

6. **Event Feedback:** Play the event the way it was meant to be played. If you find bugs, shortcuts, or glitches, report it to the Creator so they have a chance to correct it. **You can contact a Creator with feedback via DMs.**

**A message about event farming:** *Creators can only reset so often \- others may be waiting for their turn to run an event. Aim to let everyone have a chance, even if it means bowing out of an event you wanted to replay.*

*The rules below are not perfect. Following these rules to the letter can still ruin others’ experience. Be a comfy neighbor and follow the spirit of the rules so that more rules do not need to be made.*

*If you like to run events for fun, we’re a big fan of dungeoneers who bring along newbies and guide them through events. We especially like when those guides go as “Protection” (see Slayers Guild) and leave the loot for the newbies.*

7. **Anti-Farming Rules:**  
   1. **Do not run the same event more than twice per week**  
   2. **Do not run the same event two resets in a row** according to its reset schedule posted in the [\#🔁-event-resets](https://discord.com/channels/820120530107367435/822500121726943232) channel.  
   3. **No event may be soloed more than 3 times maximum per era.** This includes solo events and solo runs of multiplayer events (See Rule \#8b). Practicing parkour doesn’t count towards the limit if you take no loot.

8. **How to Solo a Creator Event:**  
   1. **Prearranged Solo:** DM the Creator of the event asking for permission to run it solo. They may restrict you from taking some or all of the loot.  
   2. **Solo Due to No Interest:** Put in a good faith effort first to recruit players:  
* Shout in **both** in-game chat and Discord in [\#🤝-looking-for-group](https://discord.gg/kqzSmjMn8N).  
* You must mention the specific event; "LFG for any event' doesn't count  
* You must use the @Events role ping  
* If no one responds after 15 minutes **and** the event doesn’t say No Soloing in either the Discord listing or on signs, you may start solo.

9. **First Timer Protections:**  
   1. **First Time Thursdays:** Every Thursday, half the players in an event group must be players **who haven't yet run that event this era**. **If you can't find or make a valid group, you may not run the event.**  
      Starts Thursday 9AM CEST // 3AM EST // 12 AM PST and lasts 24 hours.  
   2. **Looking For First Time:** If running an event for the first time this era, you can type **@Events LFFT for “*Event Name*”** in [\#🤝-looking-for-group](https://discord.gg/kqzSmjMn8N). In an LFFT posted run players who have already run the event this era must not take loot, and must give priority to first timers if the group is full.

## How to read this

- `repo_created_at` is the first commit in which the current path appears.
- `repo_updated_at` is the most recent commit that changes the current path.
- Scores combine update recency, touch frequency, active lifespan, and recent burst activity, then apply a capped logarithmic size weight.
- The temporal heatmap uses first-seen era on one axis, last-updated era on the other, size for cell area, and touches for color.
- Clone-local filesystem timestamps and ignored/private files do not affect these reports.

# Era-save classification

[`classification.json`](classification.json) is a preserved classification artifact from the Era
save analysis. It is reference data, not an active runtime component.

Use the current field program in [`../fieldlab/README.md`](../fieldlab/README.md) for live experiment
state and [`../docs/lumberjacks-native-runtime-era-save-plan.md`](../docs/lumberjacks-native-runtime-era-save-plan.md)
for the broader era-save plan.

This directory remains separate so a large or one-off analytical result is not mistaken for source
code or silently folded into a generated dataset.

# Quest picker design exploration

This directory is a preserved design snapshot for the quest-selection interface. It contains the
discovery HTML, its supporting JavaScript, a thumbnail, and an uploaded visual reference.

It is historical design evidence, not the canonical runnable picker. For the current flow, use:

- [`../recipes/README.md#quest-catalogs`](../recipes/README.md#quest-catalogs) for the generating recipe
- [`../data/processed/quest-picker.html`](../data/processed/quest-picker.html) for the generated picker
- [`../docs/quest-vertical-slice-architecture.md`](../docs/quest-vertical-slice-architecture.md) for
  the end-to-end architecture

Keep this directory intact when revisiting the interface so the original exploration remains
distinguishable from production outputs.

# Example input — the Slayers rank chart as it exists today

This is how the ladder actually lived before the recipe: a hand-drawn flowchart, read left to right,
one row of boxes per rank-up, the rank name in the last box. Reproduced here as text so you can see
the "before". `example-output.json` is the "after".

**Rank 1 → Thrall**
[ 5 contracts ] → [ 5 general summons: Know your Enemies I & II, Intermediate Killer, Clash of
Titans, Air Drop ] → [ Kneel at HQ and request Viking Trainee ] → Rank to Thrall

**Rank 2 → Thegn**
[ 9 Contracts, 2 Bounties ] → [ 10 summons: Blacksmith I, Bonking Bonemass, Crusader, Fenrir's
Children, Helpful Hunter ] → [ Thrall Duel or Mercy Rule ] → Rank to Thegn

**Rank 3 → Jarl**
[ 15 Contracts, 5 Bounties, 10 Class Summons ] → [ Death from Above, Full House, Swift Sneak, Self
Sufficient, Silent Killer, Gjallarhorn Come, Back Stabber, Cold Shot, Falling Hard, Skilled Rogue ]
→ Rank to Jarl

(There is also a starting rank, **Recruit**, for anyone who has joined but not yet ranked up.)

What the chart does NOT capture, and the recipe adds: which bot command a member pastes to submit;
a place to keep the "why" behind the numbers; and a version history so next era's changes are a diff,
not a redraw. What the chart drifts on — the image gets stale — the recipe fixes by regenerating.
